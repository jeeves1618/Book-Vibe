Spring & Microservices Study Planner Course 1 + Course 2

Start date: 19 September · Duration: about 12 weeks (excluding buffer weeks) · Buffer weeks after every 3 normal weeks

Courses Overview & Links

Course 1: Spring & Spring Boot + JPA + Hibernate — covers Spring Core, IoC/DI, AOP, Spring MVC, Spring Boot auto-configuration, Spring Security, Spring JDBC, Spring Data JPA, REST services, Actuator, logging, profiles, etc.
Go to Course 1 on Udemy

Course 2: Master Microservices with Spring, Docker & Kubernetes — covers microservices architecture, Spring Cloud (config, service discovery, API Gateway), resilience (circuit breakers), observability, security (OAuth2 / JWT), containerization (Docker), Kubernetes & Helm.
Go to Course 2 on Udemy

This planner allocates first to Course 1 topics, then transitions into Course 2’s microservices and infrastructure content, skipping overlapping fundamentals.

  • Implement logging/aspect advice using AOP

Week 13 Jan 29 — Feb 4 Course 2: Kubernetes Basics, Deploy Microservices to K8s, ConfigMaps & Secrets Learn Kubernetes deployment, config management, secrets
